# La Oca Musical

Juego educativo interactivo de La Oca Musical - Convertir el clásico juego de mesa en una aplicación web interactiva con React.

## Estructura del proyecto

- **/tablero** - Tablero del juego
- **/simbolos-musicales** - Símbolos musicales personalizados
- **/src** - Código fuente de la aplicación React
